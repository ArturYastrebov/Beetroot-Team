Задача:
Используя asyncio cоздать клиент и сервер. Создать тест для проверки заданной функциональности
Клиент:
0) Клиент шлет реквест на подключение к базе, сервер ждет
запрос на подключение к базе перед выполнением любых других реквестов
!реальный коннект к базе не нужен, только имитация с помощью принта
1) Шлёт реквест
2)Ожидает респонс от сервера
Сервер:
1)Ожидает любой входящий пакет и превращает его в ожидаемую репрезентацию
2)Ожидает случайный таймаут от 2 до 5 секунд
3)Реквесты принимаются и обрабатываются асинхронно.
Это значит, что если клиент ~одновременно отправит два реквеста на которые выпадает
таймаут 2 и 3 секунды соответственно то первый респонс будет получен спустя ~2
секунды после отправки, а второй спустя ~3
&& - разделитель параметров
%% - разделитель сущностей
Формат пакета JSON:
Пакет содержит:
request_id - уникальный идентификатор реквеста
Data - поле с данными которые нужно конвертить в ожидаемый формат
Структура data
{Entity_name}&&{param1name}&&{param1value}&&{param2name}&&{param2value}...%%{En
tity_name}...
Пустые элементы не должны входить в конечный результат
Пример:
Формат: JSON с полями request_id и data

DB_REQ = {
	"request_id": "01",
	"data": "db_connect",
}
DB_RESP = {
	"request_id": "01",
	"data": "db_connected"
}
REQ =
{
"request_id":"01",
"data":"Device1&&name&&dev_name&&temperature&&25%%Device2&&name&&dev_name&&temperature&&4"}

RESP =
{
'request_id': '01',
'data': {
'Device1': {'name': 'dev_name',
	   'temperature': '25',
	   reaction: "Hot & Sweaty!",
'Device2': {'name': 'dev_name',
	   'temperature': '4',
	   reaction: "Icy Cool!"},
}
}
Сервер имитирует системы термостата умного дома и формирует фидбек по температуре в запросе
в формате температура -> реакция:
До 10°C включительно - Icy Cool!
11°C - 15°C - Chilled Out!
16°C - 19°C - Cool Man!
20°C - 22°C - Too Warm!
Больше 22°C - Hot & Sweaty!

2. С помощью pytest написать параметризованный тест, который исходя из условия отправляет на сервер реквест с соответствующей температурой и подключается к базе для выполнения этих реквестов
!один раз за весь тест
